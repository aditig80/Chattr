## 🚀 Features
✅ Real-time messaging with **Socket.IO**  
✅ User authentication with **JWT**  
✅ Online/offline status indicators  
✅ Beautiful, responsive **Tailwind CSS** UI  
✅ User avatars & message seen status  
✅ Group and private chats

cd ../frontend
